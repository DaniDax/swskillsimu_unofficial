SoulWorker Skill Simulator

The skill simulator is made (mainly) based on the information from SoulWorker in Japan region.

This skill simulator require javascript turned on.
The skill simulator should work on most of modern browsers. I just tested on: Firefox v62, Chrome v69, Edge.

As of now, the simulator video has skill preview video (which require Promise object).

Full change-log can be found here: Github Repository commit log.
